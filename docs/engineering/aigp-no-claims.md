# AI Grand Prix Non-Claims

Until specific claim issues are opened and promoted, this repo does not claim:

- winning ability;
- state-of-the-art performance;
- speedup;
- reliability rate;
- official submission readiness;
- physical-drone safety;
- sim-to-real transfer;
- generalization to Round 2;
- accurate velocity telemetry handling;
- validated learned policy behavior;
- validated world-model advantage;
- AGIBOT benchmark performance;
- Go2 dataset superiority for training;
- WorldForge runtime control.

Allowed early claims:

- repository files exist;
- local validation commands passed;
- a fixture is deterministic;
- a source was checked on a specific date;
- a claim is explicitly limited to a named simulator, fixture, command, or artifact.
